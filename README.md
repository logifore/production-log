# 制片日志 2.0

制片日志是面向影视、广告与内容制作团队的本地优先制片协作工具。它将项目资料、拍摄日程、通告、人员、预算和现场记录集中在同一工作台，并通过 Web、Cloudflare Worker/D1 与微信小程序覆盖桌面、移动浏览器和现场协作场景。

## 核心能力

- 项目、拍摄日、进度、人员、预算、通告、场地路线与资料归档管理。
- Excel、PDF、图片和项目备份的本地导入导出。
- 本地优先的数据保存、快照恢复与加密备份。
- 基于 Cloudflare Worker/D1 的项目同步、成员权限和网页配对。
- 微信小程序端的项目浏览、现场记录与跨端同步。

## 发布内容

此仓库只保留可发布文件，并按目标平台拆分：

- `web/`：上传至 Web 静态托管平台的 PWA 成品。
- `cloudflare-worker/`：部署 Cloudflare Worker 的源码、D1 迁移与必要共享模型。
- `miniprogram/`：导入微信开发者工具后上传的小程序成品。

## 部署顺序

1. 部署 `cloudflare-worker/`，创建 D1 数据库并配置 Worker 域名、跨域来源和密钥。
2. 使用正式 Worker API 域名构建并发布 Web 端。
3. 使用同一 API 域名构建小程序，填写微信 AppID 后在微信开发者工具中上传。

Web 和小程序的跨端同步依赖已部署的 Worker API。请勿将 D1 数据库 ID、AppID、访问令牌或其他密钥提交到仓库。
